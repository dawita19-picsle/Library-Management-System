# 📚 Library Management System

A comprehensive and interactive desktop application designed for efficient academic resource tracking and management. This system streamlines the daily operations of a library, making it easy to manage books, student records, and borrowing activities.

## ✨ Key Features
* Book Management: Add, update, and remove books from the library database.
* Student Tracking: Register students and manage their borrowing privileges.
* Issue & Return Tracking: Easily issue books to students and process returns with automated record updates.
* Interactive Dashboard: A user-friendly graphical interface designed for seamless navigation and quick access to data.
* Secure Database Integration: Reliable data storage and retrieval using a relational database.

## 💻 Tech Stack
* Language: Java (Object-Oriented Programming)
* Framework (GUI): JavaFX
* Database: MS SQL Server (MSSQL)
* IDE: IntelliJ IDEA
